* Question
In this exercise we implement the method just described for interpreting internal definitions. We assume that the evaluator supports ~let~ (see exercise 4.6).

a. Change ~lookup-variable-value~ (section 4.1.3) to signal an error if the value it finds is the symbol ~*unassigned*~.

b. Write a procedure ~scan-out-defines~ that takes a procedure body and returns an equivalent one that has no internal definitions, by making the transformation described above.

c. Install ~scan-out-defines~ in the interpreter, either in ~make-procedure~ or in ~procedure-body~ (see section 4.1.3). Which place is better? Why?

  (define (lookup-variable-value var env)
    (let ((val (val-binding (first-binding (env-loop var env)))))
      (if (eq? val '*unassigned*)
          (error "Unnassigned variable -- LOOKUP-VARIABLE-VALUE" var)
          val)))

*** b-c. ~scan-out-defines~
Creating the internal block is a one-time task, so it is better to do it in ~make-procedure~.

~scan-out-defines~ creates a ~let~ which creates a ~lambda~ which ends up producing a procedure whose body is processed by ~scan-out-defines~ -- therefore, it is important to break the cycle when no ~defs~ are found.

#+begin_src scheme :tangle 4.16.mceval.apply.compound.scm
  (define (scan-out-defines body)
    (let* ((def?
             (lambda (exp)
               (tagged-list? exp 'define)))
           (make-exp
            (lambda (defs rest)
              (make-let (map (lambda (x)
                               (list (definition-variable x)
                                     '(quote *unassigned*)))
                             defs)
                        (append (map (lambda (x)
                                       (make-assignment (definition-variable x)
                                                        (definition-value x)))
                                     defs)
                                rest)))))
      (let loop ((exps body)
                 (next (lambda (defs rest)
                         (if (null? defs) rest ;; Stop recursing
                             (list (make-exp defs rest))))))
        (if (null? exps) (next '() '())
            (let ((exp (car exps)))
              (loop (cdr exps)
                    (lambda (defs rest)
                      (if (def? exp)
                          (next (cons exp defs) rest)
                          (next defs (cons exp rest))))))))))

  (define (make-procedure parameters body env)
    (list 'procedure parameters (scan-out-defines body) env))
#+end_src
