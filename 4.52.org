* Question
:PROPERTIES:
:header-args:scheme: :eval no
:END:
Implement a new construct called ~if-fail~ that permits the user to catch the failure of an expression. ~If-fail~ takes two expressions. It evaluates the first expression as usual and returns as usual if the evaluation succeeds. If the evaluation fails, however, the value of the second expression is returned, as in the following example:

#+begin_src scheme
  ;;; Amb-Eval input:
  (if-fail (let ((x (an-element-of '(1 3 5))))
             (require (even? x))
             x)
           'all-odd)
  ;;; Starting a new problem
  ;;; Amb-Eval value:
  all-odd
  ;;; Amb-Eval input:
  (if-fail (let ((x (an-element-of '(1 3 5 8))))
             (require (even? x))
             x)
           'all-odd)
  ;;; Starting a new problem
  ;;; Amb-Eval value:
  8
#+end_src

* Answer
** Implementation
:PROPERTIES:
:header-args:scheme: :eval no
:END:
#+begin_src scheme :tangle 4.52.ambeval.eval.if-fail.scm
  (define (install-if-fail-package)
    (put 'eval 'if-fail
         (lambda (exp)
           (let ((pproc (analyze (if-predicate exp)))
                 (cproc (analyze (if-consequent exp))))
             (lambda (env succeed fail)
               (pproc env
                      succeed ;; Pass success branch
                      (lambda () ;; Divert to failure branch
                        (cproc env
                               succeed
                               fail)))))))
    'ok)
#+end_src

#+begin_src scheme :tangle 4.52.ambeval.scm
  ;; Inherit
  (load "4.51.ambeval.scm")
  ;; Expression types
  (load "4.52.ambeval.eval.if-fail.scm")
  ;; Setup
  (install-if-fail-package)
#+end_src
